Curly Invention: MLOps Platform

Welcome to the Curly Invention MLOps project. This site provides an overview of the main components and structure of our MLOps pipeline.

MLOps Structure

  • Data Ingestion: Scripts and configs for data collection and ingestion.
  • Dataset Versioning: Tools and metadata for dataset versioning.
  • Model Registry: Model tracking and registry files.
  • ML Code: ML model code, training, and evaluation scripts.
  • Software Engineering: Supporting utilities, CI/CD, and infrastructure code.
